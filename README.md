# Projet : Site Web de Musiqu SoundWave

## Description
Ce projet est un site web de musique conçu à partir d'une maquette reproduite sur Figma. Il a été réalisé en suivant une vidéo tutorielle. Le site utilise HTML et CSS pour les interactions et la mise en page.la reproduction de ce site n'a pas été un grand succée pour moi j'ai rencontré beaucoup de difficultés plus particulièrement sur la responsivité.

## Fonctionnalités
- Présentation d'une page d'accueil avec une introduction au site.
- Intégration d'une galerie de photos pour mettre en valeur les artistes ou les événements.
- Formulaire client pour l'inscription ou la réservation d'événements.
- Intégration des liens vers les réseaux sociaux utilisés par le site de musique.

## Technologies Utilisées
- HTML
- CSS

## Installation
1. Clonez ce dépôt sur votre machine locale :
    ```bash
    git clone <url-du-depot>
    ```
2. Ouvrez le fichier `index.html` dans votre navigateur web.
